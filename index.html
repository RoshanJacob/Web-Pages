HTML (HyperText Markup Language) is mainly used to create websites. Any website on the web written in html will have a head and body tag. Along with html, css can be used to decorate and style webpages. The < style > < / style > tag can also be used in the head tag to style elements.

Attributes

Attributes are special pre-defined words used inside the opening tag of a html element. Example â†ª <body bgcolor = "yellow"> . They provide additional information about the element, such as it's style, width, colour, shadow etc. Specific elements have their own attributes that apply to them. 
                            
Line breaks and the paragraph tag

        Line breaks, denoted by the < br > tag, is used to take a html element to the next line
        with a break. Block-level elements show this property automatically while with inline elements
        it is necessary to define it to take it to the next line.

        The paragraph tag (< p >) is used to denote a paragraph in html. A paragraph always starts
        on a new line and browsers usually put a margin before and after a paragraph.
